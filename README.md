# CTF-Writeups
This repository contains write-ups of Capture The Flag (CTF) challenges I have completed while participating cybersecurity, web exploitation, digital forensics, and security analysis.
These exercises help strengthen real-world skills used by security analysts such as:
- Web application testing
- Network analysis
- Log investigation
- Vulnerability Identification
- Exploitation techniques

# Platforms
Challenges in this repository come from:
- TryHackMe
- SANS BootUp CTF
- Cyber Skyline NCL
- WiCyS CTF


Each challenge folder contains:

- Challenge description
- Methodology
- Tools used
- Exploitation steps
- Flag discovery
- Screenshots when applicable

## Disclaimer

These write-ups are published **only after competitions close** and are intended for **educational purposes only**.
